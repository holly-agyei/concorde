# Concorde Integrations

This folder contains handoff notes for coding agents integrating sponsor APIs into the main YC hackathon project.

- `agentphone/README.md`: AgentPhone account snapshot, webhook behavior, local test harness, and live-call/SMS setup.
- `moss/README.md`: Moss role in the hero demo, credentials, seed corpus, CLI/Python setup, and agent tool shape.
- `moss/moss_seed.json`: Small semantic-search corpus for the SFO wrong-terminal demo.

Keep real API keys in local `.env` files only. Do not commit secrets.
